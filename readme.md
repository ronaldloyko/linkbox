# Linkbox

### Link Manager For Android

</div>

## Introduction

Linkbox is a **free and open source** application for Android to **manage all your links**.

## Features

- Organize links using **folders**
- **No internet** connection required
- **No ads** and **no tracking**
- True black theme as **dark mode**
- Easy **import and export** of all data and settings using simple JSON files
- Link **sharing**
